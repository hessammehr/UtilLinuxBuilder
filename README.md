# UtilLinuxBuilder

[![Build Status](https://travis-ci.org/hessammehr/UtilLinuxBuilder.svg?branch=master)](https://travis-ci.org/hessammehr/UtilLinuxBuilder)

This repository builds binary artifacts for the UtilLinuxBuilder project. Binary artifacts are automatically uploaded to
[this repository's GitHub releases page](https://github.com/hessammehr/UtilLinuxBuilder/releases) whenever a tag is created
on this repository.

This repository was created using [BinaryBuilder.jl](https://github.com/JuliaPackaging/BinaryBuilder.jl)
